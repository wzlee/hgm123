Class Index

IYY

yy接口入口，获取到yy的其他接口和方法。

IYYAudio

语音控制接口，提供处理YY的音频信息，比如录音的控制等。

IYYChannel

频道接口，提供对频道的操作和交互。

IYYChannelAppMsg

频道应用消息接口，提供频道的应用消息发送和响应等操作，应用消息出现在应用盒子的应用消息选项卡中和公告栏下方。

IYYChannelMicList

麦序接口，提供麦序的信息和相关事件。

IYYChannelTabPage

频道tab页控制接口 。

IYYChannelUserController

频道用户控制接口。

IYYChannelUserListPopMenu

频道右键菜单接口。频道用户列表右键菜单设置和取消， 和对应的菜单事件设置 。

IYYCloud

简单存储接口。提供简单的简单存储数据服务，包括增，删，改，查的基本操作，除了 频道所有者(OW紫马)和 频道总管理(VP橙马)可以删除和修改所有数据之外，其他用户只能删除和修改自己的数据，每个用户都可以查询所有数据。

IYYCommon

公共功能原型类，提供比如事件的侦听，取消侦听等公共功能。

IYYFinance

财务接口。提供Y币扣费请求和赠送开放平台礼物接口。

IYYIM

聊天接口。提供弹出聊天对话框，弹出添加好友对话框等功能。

IYYInteraction

应用互动接口。能够提供邀请者的信息。

IYYNet

网络通讯接口。提供广播数据和接收广播数据的功能。

IYYReceptionChannel

接待频道接口，提供获取，设置，取消接待频道等功能。

IYYSecurity

安全接口。提供获取安全认证信息等功能。

IYYTempAudioSession

临时语音接口。提供创建房间、加入房间、离开房间以及在房间语音聊天的功能.

IYYUser

用户信息接口。提供获取用户的信息，接收用户信息变化事件等功能。

YYCloudFilter

简单存储条件过滤器，保存查询条件。
